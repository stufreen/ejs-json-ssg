# ejs-json-ssg

It's a working title.

## Installation

```bash
npm i @stufreen/ejs-json-ssg --save-dev
```

## Usage

```bash
npx ejs-json-ssg --config ./CONFIG.json
```

## Config file
```json
{
  "templateDir": "./path/to/templates",
  "outputDir": "./path/to/output",
  "site": "./path/to/site.json"
}
```
